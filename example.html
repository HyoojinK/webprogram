단축키 설명안내

Ctrl + Q : 태그와 태그로 둘러쌓인 코드를 접거나 펼칩니다. 단, 커서의 위치는 닫거나 펼칠 태그의 제목과 동일한 라인상에 위치시켜야 합니다.

Ctrl + Space : 미완성된 키워드를 자동으로 완성합니다.

Ctrl + Z : 코드 편집 과정에서 방금 전에 수행한 동작을 취소합니다.

Ctrl + Y, Ctrl + Shift + Z : 실행을 취소했던 동작을 다시 실행합니다.
